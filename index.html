Welkom bij IMDVERSE: De Puzzel Game

De bedoeling van het spel is om met je hand de puzzelstukken op de juiste plaats te leggen.

Richt je hand op de cursor en beweeg naar infoknop voor meer informatie over het spel of ga naar de startknop en start het spel. Indien dit niet al te best gaat kan de muis gebruikt worden Veel succes!

Deze game is gemaakt door Stef Van Boven Student aan de opleiding Interactive Media Development.

IMDVERSE is een interactieve puzzelgame die gebruik maakt van handtracking.

Er zijn 3 puzzels die je kan oplossen.

Elke puzzel bestaat uit 4 stukken.

Je kan de puzzelstukken verplaatsen door je hand te bewegen.

Als je een puzzelstuk op de juiste plaats legt, hoor je een geluid.
